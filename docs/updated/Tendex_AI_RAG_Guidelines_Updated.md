## 2. إعدادات الذكاء الاصطناعي (AI Configuration)
### 2.1 الإدارة الديناميكية (Dynamic Management)
- **قاعدة البيانات (Database-Driven):** يجب تخزين جميع إعدادات الذكاء الاصطناعي (مزود الخدمة، مفاتيح API، نقاط نهاية Qdrant) بشكل ديناميكي في جدول `AiConfiguration` الخاص بكل جهة (Tenant).
- **التشفير (Encryption):** يجب تشفير جميع مفاتيح API باستخدام AES-256 قبل حفظها في قاعدة البيانات.
- **دورة حياة الجهة (Tenant Lifecycle):** لا يتم تفعيل خدمات الذكاء الاصطناعي للجهة إلا بعد اكتمال عملية التوفير (Provisioning) بنجاح وانتقال الجهة إلى حالة تشغيلية (مثل `EnvironmentSetup` أو `Active`). يجب على `TenantProvider` التحقق من حالة الجهة قبل السماح بأي استدعاء لخدمات الذكاء الاصطناعي.
