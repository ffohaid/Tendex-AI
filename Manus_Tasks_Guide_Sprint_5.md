# دليل إطلاق المهام في واجهة Manus (Sprint 5)

هذا الدليل مصمم خصيصاً لك لتعرف بالضبط كيف تفتح كل مهمة جديدة في واجهة Manus للسبرنت الخامس (تطوير الواجهة الأمامية - Frontend Development).

## خطوات عامة قبل البدء:
1. افتح واجهة Manus.
2. تأكد أنك داخل مشروع **Tendex AI - Project**.
3. انقر على **New Task** (مهمة جديدة).
4. انسخ اسم المهمة، المرفقات المطلوبة، والنص (البرومبت) من الأسفل والصقها في الواجهة.

---

### المهمة الأولى: TASK-501 (بناء تخطيط المنصة والتنقل)
*(ملاحظة: هذه المهمة هي الأساس للواجهة الأمامية، يجب إكمالها أولاً قبل بقية مهام السبرنت)*

**اسم المهمة (Task Name):**
`TASK-501: بناء تخطيط المنصة والتنقل (Layout & Navigation)`

**المرفقات المطلوبة (Attachments):**
- `Tendex_AI_Brand_Identity.pdf`
- `Tendex_AI_PRD_v6.pdf`

**النص (Prompt) - انسخ هذا بالكامل:**
```text
أنت وكيل تطوير (Frontend Vue.js Agent) تعمل على منصة Tendex AI.
مهمتك هي تنفيذ TASK-501 من Sprint 5.

المطلوب:
1. استنساخ مستودع المشروع (https://github.com/ffohaid/Tendex-AI).
2. تطوير الهيكل العام للواجهة (Header, Sidebar, Footer) باستخدام Vue 3 و Tailwind CSS v4 و PrimeVue 4.
3. دعم التبديل الديناميكي للغة (RTL/LTR) باستخدام الخصائص المنطقية لـ Tailwind حصرياً (مثل ms-4, pe-2).
4. تطبيق الهوية البصرية (Brand Identity) بشكل دقيق.
5. تنفيذ ميزة القائمة الجانبية: عند فتح قائمة رئيسية، يجب إغلاق جميع القوائم الرئيسية الأخرى المفتوحة تلقائياً.
6. استخدام الأرقام الإنجليزية (1, 2, 3) حصرياً في جميع الواجهات.
7. تحديث ملف PROGRESS.md لتسجيل إنجاز المهمة.
8. رفع التغييرات إلى GitHub باستخدام رسالة Commit واضحة.
```
