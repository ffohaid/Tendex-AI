# Tendex AI (امباكتور AI)

منصة حكومية متطورة لإدارة المنافسات والمشتريات مدعومة بالذكاء الاصطناعي.

## هيكلة المشروع (Monorepo)

يحتوي هذا المستودع على جميع مكونات المنصة:

- `frontend/`: تطبيق الواجهة الأمامية (Vue.js 3, TypeScript, Tailwind CSS v4).
- `backend/`: تطبيق الواجهة الخلفية (.NET 10, Clean Architecture).
- `infrastructure/`: إعدادات البنية التحتية و Docker Compose.
- `docs/`: وثائق المشروع وأدلة التشغيل.

## المتطلبات الأساسية

- [Docker](https://www.docker.com/) و [Docker Compose](https://docs.docker.com/compose/)
- [Node.js](https://nodejs.org/) (الإصدار 20 أو أحدث)
- [.NET SDK](https://dotnet.microsoft.com/) (الإصدار 10)

## التشغيل المحلي

سيتم إضافة تعليمات التشغيل المحلي قريباً بعد إعداد بيئة Docker Compose في المهام القادمة.

## المساهمة

يرجى مراجعة ملفات التوجيه في مجلد `docs/` قبل البدء في التطوير، والالتزام بمعايير جودة الكود المحددة.
