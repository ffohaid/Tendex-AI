# دليل إطلاق المهام في واجهة Manus (Sprint 2)

هذا الدليل مصمم خصيصاً لك لتعرف بالضبط كيف تفتح كل مهمة جديدة في واجهة Manus للسبرنت الثاني. كل ما عليك فعله هو **نسخ ولصق** النصوص أدناه في كل مرة تبدأ فيها مهمة جديدة.

## خطوات عامة قبل البدء:
1. افتح واجهة Manus.
2. تأكد أنك داخل مشروع **Tendex AI - Project**.
3. انقر على **New Task** (مهمة جديدة).
4. انسخ اسم المهمة، المرفقات المطلوبة، والنص (البرومبت) من الأسفل والصقها في الواجهة.

### المهمة الأولى: TASK-201 (إدارة الهوية والمصادقة)
*(ملاحظة: تعتمد على TASK-105 من السبرنت الأول)*

**اسم المهمة (Task Name):**
`TASK-201: بناء خدمة إدارة الهوية والمصادقة (Identity)`

**المرفقات المطلوبة (Attachments):**
- `Tendex_AI_Architecture_v1.pdf`
- `Tendex_AI_PRD_v6.pdf`

**النص (Prompt) - انسخ هذا بالكامل:**
```text
أنت وكيل تطوير (Backend .NET Agent) تعمل على منصة Tendex AI.
مهمتك هي تنفيذ TASK-201 من Sprint 2.

المطلوب:
1. استنساخ مستودع المشروع (https://github.com/ffohaid/Tendex-AI).
2. تنفيذ نظام المصادقة باستخدام OpenIddict (OAuth 2.0 / OIDC) ودعم المصادقة الثنائية (MFA).
3. إعداد تخزين الجلسات (Sessions) في Redis.
4. إصدار Access Token بصلاحية 60 دقيقة، و Refresh Token بصلاحية 8 ساعات.
5. تحديث ملف PROGRESS.md لتسجيل إنجاز المهمة.
6. رفع التغييرات إلى GitHub باستخدام رسالة Commit واضحة.

الرجاء الالتزام بمعايير Clean Architecture و Minimal APIs.
```
