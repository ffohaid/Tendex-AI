# دليل إطلاق المهام في واجهة Manus (Sprint 4)

هذا الدليل مصمم خصيصاً لك لتعرف بالضبط كيف تفتح كل مهمة جديدة في واجهة Manus للسبرنت الرابع (تكامل الذكاء الاصطناعي و RAG).

## خطوات عامة قبل البدء:
1. افتح واجهة Manus.
2. تأكد أنك داخل مشروع **Tendex AI - Project**.
3. انقر على **New Task** (مهمة جديدة).
4. انسخ اسم المهمة، المرفقات المطلوبة، والنص (البرومبت) من الأسفل والصقها في الواجهة.

---

### المهمة الأولى: TASK-401 (بوابة الذكاء الاصطناعي)
*(ملاحظة: تعتمد على اكتمال TASK-203 من السبرنت الثاني)*

**اسم المهمة (Task Name):**
`TASK-401: بناء بوابة الذكاء الاصطناعي (AI Gateway)`

**المرفقات المطلوبة (Attachments):**
- `Tendex_AI_Architecture_v1.pdf`
- `Tendex_AI_PRD_v6.pdf`

**النص (Prompt) - انسخ هذا بالكامل:**
```text
أنت وكيل تطوير (Backend .NET Agent) تعمل على منصة Tendex AI.
مهمتك هي تنفيذ TASK-401 من Sprint 4.

المطلوب:
1. استنساخ مستودع المشروع (https://github.com/ffohaid/Tendex-AI).
2. تطوير واجهة موحدة (AI Gateway) للاتصال بنماذج الذكاء الاصطناعي المختلفة (OpenAI, Claude, نماذج محلية).
3. بناء آلية لجلب إعدادات النماذج ومفاتيح API من جدول `AiConfiguration` في قاعدة البيانات.
4. تطبيق التشفير (AES-256) لمفاتيح API في قاعدة البيانات، وفك تشفيرها في الذاكرة (In-memory) فقط أثناء التنفيذ.
5. دعم التبديل الديناميكي بين النماذج.
6. تحديث ملف PROGRESS.md لتسجيل إنجاز المهمة.
7. رفع التغييرات إلى GitHub باستخدام رسالة Commit واضحة.
```
